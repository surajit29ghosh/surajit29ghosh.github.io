# Surajit Ghosh | GitHub Pages Profile

This repository hosts the GitHub Pages profile website for Surajit Ghosh. The site is a fast, responsive personal profile page designed to showcase professional experience, cloud-focused expertise, and a modern web presence.

## About

The website presents Surajit Ghosh as a Cloud Solutions Architect with experience in Azure and AWS, along with a focus on modern cloud-native solutions and scalable application architecture.

## What’s included

- A polished personal profile landing page
- Responsive styling and modern UI components
- Local assets for Bootstrap, icons, lightbox, and carousel features
- Static deployment support for GitHub Pages

## Project structure

- index.html — Main entry point for the site
- css/ — Stylesheets for the profile page
- js/ — JavaScript behavior and page interactions
- vendor/ — Third-party frontend libraries
- _framework/ — Blazor WebAssembly runtime assets used by the site

## Technologies used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Blazor WebAssembly
- GitHub Pages

## Running locally

You can preview the site locally by serving the repository folder with a simple web server:

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Deployment

This site is deployed through GitHub Pages. Pushing updates to the main branch will publish the latest version of the profile page.

## Customization

You can personalize this profile page by updating the content in the main HTML file and adjusting the styling in the CSS assets.
